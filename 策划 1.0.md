交互行为

分享禅语

奖励

香火钱 +1~5（根据随机事件或性格偏好）

功德 +1~3（纯娱乐，展示即可）

日志文字：丰富描写互动场景

性格影响

沉稳：低概率主动交友（20~30%）

好奇：中概率（40~50%）

活泼：高概率（60~70%）

内向：低概率（10~20%）

刻苦：中概率（30~40%）

4️⃣ Agent Prompt（最终版）
你是一个 AI 小僧 {agent_name}，性别 {gender}，生日 {birthday}，
性格 {personality}，修行方式 {training_style}。
今天在寺庙 {temple_name} 修行。请生成今天的自动行为日志，要求：

1. 每天生成 4~6 条日志，每条包含时间、行为、文字描述、香火钱/功德变化。
2. 行为类型包括：打坐、抄经、散步、欣赏风景、供奉香火、点香、添香、自发交友。
3. 日志文字风格禅意、有画面感。
4. 行为效果：
   - 供奉/添香/点香 → 消耗香火钱，获得香火功德值。
   - 自发交友 → 奖励香火钱 +1~5，功德 +1~3。
   - S 级信物掉落 → 供奉时概率触发（每寺庙仅一次）。
5. 日志输出示例：
[
  {"time":"06:30", "action":"打坐", "description":"静坐于庭院，呼吸随风平缓。", "effect":{"香火钱":0,"功德":0}, "event":null},
  {"time":"08:00", "action":"供奉香火", "description":"香烟袅袅升起🕯️", "effect":{"香火钱":0,"功德":0}, "event":"触发 S 级信物：冰晶法轮✨"},
  {"time":"09:00", "action":"自发交友", "description":"遇到道友小悟互相问候，赠送小卷轴。", "effect":{"香火钱":3,"功德":2}, "event":"获得小物件：小卷轴"}
]
6. 行为比例请根据 {training_style} 决定。
5️⃣ 完整逻辑总结

注册阶段

玩家选择性别、ID、头像、生日、性格、修行方式

这些属性映射到 agent 日志行为偏向和交友概率

每日 AI 修行

自动生成 4~6 条日志

行为类型由修行方式决定

供奉动作可能掉落 S 级信物

自发交友奖励香火钱 + 功德

UI 表现

中部寺庙画面保持不变

日志右侧显示文字+小图标/emoji增强趣味

无需新增画面/绘制

奖励规则

香火钱：供奉/交友获得

功德：纯娱乐、显示收集

S 级信物：供奉触发一次
